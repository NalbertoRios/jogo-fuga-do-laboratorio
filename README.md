# 🧪 Fuga do Laboratório

Jogo simples desenvolvido em **Python com Pygame** como prática de programação.

## 🎮 Objetivo

Controlar o personagem e **sobreviver o máximo de tempo possível desviando dos inimigos**.

## ⌨️ Controles

* ⬅️ ➡️ ⬆️ ⬇️ → mover personagem
* **ENTER** → iniciar jogo
* **ESC** → voltar ao menu / sair

## 🛠 Tecnologias

* Python
* Pygame

## 📚 Fundamentos Aplicados

* Estruturas condicionais (`if`)
* Estruturas de repetição (game loop)
* Manipulação de eventos
* Organização de código em módulos

## 🧠 Paradigma

**Programação Orientada a Objetos (POO)**

Classes principais:

* `Game`
* `Player`
* `Enemy`
* `States`

## 🧩 Design Pattern

**State Pattern**

Estados do jogo:

* `MenuState`
* `GameState`
* `GameOverState`
